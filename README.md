## What it is

Recipe API is a clean, well-documented REST API that gives developers instant access to a large library of recipes,
ingredients, and nutritional information, without the friction of legacy food data providers.

- **50,000+ recipes** across multiple cuisines, difficulties, and meal types
- **10,000+ ingredients** with categories and nutritional breakdowns
- **Dietary filters** for vegan, gluten-free, allergen-aware, and more
- **Multi-language** responses (English, French, Spanish)
- **Free tier** with 500 requests per month, no credit card required
- **Simple auth** using Bearer tokens, JSON responses, sensible rate limits

## Quick start

```bash
curl "https://recipeapi.io/api/v1/recipes?cuisine=italian&limit=5" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Full documentation at [recipeapi.io/docs](https://recipeapi.io/docs).
